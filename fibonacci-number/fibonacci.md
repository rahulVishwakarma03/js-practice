
# WAP to get nth fibonacci number.
fibonacci series : 0,1,1,2,3,5,8,13

| input | output |
------------------
|   1   |   0    |
|   2   |   1    |
|   3   |   1    | (1+0)
|   4   |   2    | (1+1)
|   5   |   3    | (2+1)
|   6   |   5    | (3+2)
|   7   |   8    | (5+3)

Logic :

Current Fibonacci Value = previous1 fibonacci number + previous2 fibonacci number
